Check available snippets:

bash

bash

ğŸ“– Usage
Basic Commands
List All Available Snippets
bash
devgen list
Get a Specific Snippet
bash
devgen get <snippet-name>
Examples:

bash
devgen get react-component.js
devgen get express-api.js
devgen get mongodb-model.js
Generate Folder Structure
bash
devgen gen-folder
Generate Full Express App
bash
devgen express-setup [project-name]
Example:

bash
devgen express-setup my-api-project
Advanced Usage
Specify Output Directory
bash
devgen get react-component.js --output ./src/components
Generate with Custom Name
bash
devgen get react-component.js --name UserProfile
Generate with TypeScript
bash
devgen get react-component.js --typescript
Show Help
bash
devgen --help
devgen get --help

Q: The command 'devgen' is not recognized
A: Ensure you installed globally:

bash
npm install -g dev-genius-tools
Q: Can I use this with TypeScript?
A: Yes! Use the --typescript flag:

Q: How do I add my own snippets?
A: Currently, custom snippets require modifying the source code. We're working on a plugin system for future versions.
